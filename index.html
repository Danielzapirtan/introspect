Instrucțiuni pentru publicare pe GitHub Pages

  1. Completează toate câmpurile obligatorii din formular
  2. Apasă "Generează website" pentru a descărca fișierul index.html
  3. Încarcă fișierul în repository-ul tău GitHub (crează unul nou dacă nu ai)
  4. Activează GitHub Pages din setările repository-ului (Settings → Pages)
  5. Alege branch-ul principal (main/master) și salvează setările
  6. Website-ul tău va fi disponibil la adresa: https://[username].github.io/[repository]
